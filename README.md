A web-based Campus Safety System designed to improve student security by providing quick emergency support, incident reporting, and location-based assistance within the campus.

## 🚀 Features

🔐 User Authentication (Login & Registration)
👤 Student Dashboard
🚨 Emergency SOS Button
📝 Incident Reporting
📍 Live Location Sharing
📞 Emergency Contact Management
📊 Admin Dashboard
📋 View and Manage Reports
📱 Responsive Design
